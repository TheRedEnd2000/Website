Frequently Asked Questions

  • Where can I download all the plugins?

    You can download all the plugins on SpigotMc.org.

  • Where can I find help if a plugin does not work?

    You can join the discord or send a support message in the support tab.

  • How do I install the plugins on my server?

    1. Go to your server files and find the plugins folder
    2. Download the plugin on SpigotMC.org
    3. Put the plugin in your plugins folder
    4. Restart your server
    5. Do /plugins and see if the plugin is listed
    Note: If the plugin is not listed please look at the requirements.
